RL Tracker

This web app was specifficaly targeted for a friend which is focused on the game Rocket League, trying to master the game and become better!

So this is how it works:

  1. You select a category from the given options when clicked on the burger menu.
  2. After that you get to choose if you would like to train (in Rocket Leage) on hard mode, medium or easy
  3. When you have selected the mode, you can no start training (in Rocket League) and assign every shot on goal or trick done by either pressing plus(adding a score)
    miss(ex: missed a goal) or in case of a illegal goal you can subtrack the attempt
  4. After you're done with the training you take a screen shot by tapping on the righ sided button so you can save them locally
  5. For the next time you want to train your skills you can easily track if you've become better by uploading those screenshot by tapping on the left sided button
    6. (Keep in mind, it only supports .PNG format ;D)
